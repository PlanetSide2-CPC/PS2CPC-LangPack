# 自动语言文件转换工具（PS2）

> Weblate 的文件格式转换支持

获取源翻译文件并转换成新的文件格式，保留内容。

## 项目功能

- 支持将原 dat 文件格式转换为 xml（已完成）或 json（未开发）。

## 使用说明

包含如何使用该程序注意事项。

### 安装

在 `setup.py` 目录下运行以下命令安装项目：

```shell
pip install .
```

### 运行

使用以下指令运行程序

```shell
python -m spinbot
```
